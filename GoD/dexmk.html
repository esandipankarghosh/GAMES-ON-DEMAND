Deus Ex- Mankind Divided

Mankind Divided is built using the Dawn Engine. We expect this next Deus Ex game to require some pretty high system specs, with a need for a high end modern graphics card to reach 1080p and 60 frames per second. This will be something like an Nvidia GTX 970 or AMD R9 390 for the recommended GPU. The Deus Ex Mankind Divided processor would be something like the Intel Core i7-3770 or the trusty top end AMD FX8350. System memory will likely need 16GB. But Deus Ex often comes out with accessible low end minimum specs, that enable a broad range of gamers to be able to run it. More information on this as we get closer to release.
Its 2029, augmented humans have been categorised as social outcasts.Adam Jenson operates as a covert ops specialist for a world that despises his kind.
